## Cấu Trúc Thư Mục

Dự án được tổ chức thành các thư mục với chức năng cụ thể như sau:

| Thư mục | Mô tả |
| :--- | :--- |
| 📁 `Checkpoint_PDF/` | Nơi lưu trữ các file PDF đang trong quá trình chỉnh sửa, cập nhật thường xuyên. |
| 📁 `Codes/` | Chứa toàn bộ mã nguồn (source code) của dự án. Bao gồm các script làm sạch dữ liệu (data preprocessing/cleaning) và mã nguồn huấn luyện mô hình (training scripts cho YOLO, LSTM,...). |
| 📁 `Official/` | Chứa các tài liệu chính thức của đồ án. |
| 📁 `Output/` | Nơi lưu trữ kết quả đầu ra, chủ yếu là các file trọng số của mô hình sau khi đã hoàn tất quá trình huấn luyện. |
| 📁 `References/` | Chứa các tài liệu tham khảo khoa học, các bài báo khoa học, sách, hoặc các tài liệu nền tảng liên quan đến dự án. |
| 📁 `Thesis/` | Chứa toàn bộ nội dung của quyển báo cáo nghiên cứu (thesis/report). Bao gồm các file mã nguồn LaTeX (`.tex`), thư mục chứa hình ảnh minh họa (`images/`), và các file cấu hình liên quan đến việc định dạng quyển đồ án. |

## Tuyên bố bản quyền 
Dự án này là sản phẩm trí tuệ phục vụ cho mục đích nghiên cứu học thuật và là sản phẩm khóa luận tốt nghiệp của nhóm thực hiện. 
- **ĐƯỢC PHÉP:** Xem, tham khảo logic code, tìm hiểu kiến trúc mô hình và học hỏi cách trình bày báo cáo.
- **KHÔNG ĐƯỢC PHÉP:** Sao chép nguyên vẹn mã nguồn hoặc báo cáo để tái sử dụng cho các môn học, đồ án tốt nghiệp, hoặc các mục đích thương mại mà không có sự đồng ý hoặc trích dẫn rõ ràng.
